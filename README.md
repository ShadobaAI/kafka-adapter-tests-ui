# 1С: Адаптер Kafka — UI-Тесты

![Платформа](https://img.shields.io/badge/1С-8.3.21+-blue)

Набор UI-тестов для проверки поведения [1С: Адаптер Kafka](https://github.com/ShadobaAI/kafka-adapter).

## Назначение

Репозиторий содержит тесты для проверки изменений адаптера:

- **UI-тесты Vanessa Automation** — сценарии в каталоге `features`, которые проверяют поведение адаптера через интерфейс пользователя;

> Репозиторий предназначен только для разработки и проверки изменений адаптера. В промышленную эксплуатацию не поставляется.

## Инструменты

- [Vanessa Automation](https://pr-mex.github.io/vanessa-automation/) — UI-тесты для проверки пользовательских сценариев.
